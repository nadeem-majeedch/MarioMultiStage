# Mario Game Project

This is a simple Mario-style game implemented in Python using Pygame.

## Features
- Basic platformer mechanics
- Multiple stages
- Simple enemy AI

## Requirements
- Python 3.10+
- Pygame

## Installation
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the game: `python MarioMultiStage.py`

## Game Controls
- Arrow keys: Move
- Space: Jump
- Shift: Run
